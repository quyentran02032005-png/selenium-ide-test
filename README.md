# Kiểm tra Selenium IDE

- Công cụ: Selenium IDE (Firefox)
- Hình thức: Test tự động
- Tổng số test case: 92

## Mô tả
- Các test case được xây dựng bằng Selenium IDE
- Do số lượng test case lớn, test được chia thành nhiều file `.side`
- Các file `.side` có thể import trực tiếp vào Selenium IDE để chạy lại

## Minh chứng
- Kết quả test được chụp màn hình khi chạy test tự động
